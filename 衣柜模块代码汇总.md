## 一、后端代码

### 1. WardrobeController.java

路径：`Wear_project_demo/src/main/java/com/campus/outfit/controller/WardrobeController.java`

```java
package com.campus.outfit.controller;

import com.campus.outfit.dto.resp.Result;
import com.campus.outfit.entity.Clothing;
import com.campus.outfit.mapper.ClothingMapper;
import com.campus.outfit.util.MinioUtil;
import org.springframework.web.bind.annotation.*;
import org.springframework.web.multipart.MultipartFile;

import java.util.HashMap;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/wardrobe")
public class WardrobeController {

    private final MinioUtil minioUtil;
    private final ClothingMapper clothingMapper;

    public WardrobeController(MinioUtil minioUtil, ClothingMapper clothingMapper) {
        this.minioUtil = minioUtil;
        this.clothingMapper = clothingMapper;
    }

    /**
     * 上传衣服
     */
    @PostMapping("/upload")
    public Result upload(@RequestParam("file") MultipartFile file,
                        @RequestParam("type") String type,
                        @RequestParam("color") String color,
                        @RequestParam("style") String style,
                        @RequestParam("season") String season,
                        @RequestParam("description") String description) throws Exception {
        // 上传图片到campus-outfit桶的myclothes目录
        String fileName = minioUtil.upload(file, "campus-outfit", "myclothes/");
        String url = minioUtil.getUrl(fileName, "campus-outfit", "myclothes/");

        // 保存衣服信息到数据库
        Clothing clothing = new Clothing();
        clothing.setUserId(1); // 暂时使用默认用户ID，后续可从登录信息中获取
        clothing.setFileName(fileName);
        clothing.setUrl(url);
        clothing.setType(type);
        clothing.setColor(color);
        clothing.setStyle(style);
        clothing.setSeason(season);
        clothing.setDescription(description);
        clothing.setCreateTime(System.currentTimeMillis());

        clothingMapper.insert(clothing);

        return Result.success("上传成功", Map.of("id", clothing.getId(), "url", url));
    }

    /**
     * 获取用户衣柜列表
     */
    @GetMapping("/list")
    public Result list(@RequestParam("userId") Integer userId) {
        List<Clothing> clothingList = clothingMapper.selectByUserId(userId);
        return Result.success("获取成功", clothingList);
    }

    /**
     * 根据类型获取衣服
     */
    @GetMapping("/listByType")
    public Result listByType(@RequestParam("userId") Integer userId, @RequestParam("type") String type) {
        List<Clothing> clothingList = clothingMapper.selectByUserIdAndType(userId, type);
        return Result.success("获取成功", clothingList);
    }

    /**
     * 根据季节获取衣服
     */
    @GetMapping("/listBySeason")
    public Result listBySeason(@RequestParam("userId") Integer userId, @RequestParam("season") String season) {
        List<Clothing> clothingList = clothingMapper.selectByUserIdAndSeason(userId, season);
        return Result.success("获取成功", clothingList);
    }

    /**
     * 根据风格获取衣服
     */
    @GetMapping("/listByStyle")
    public Result listByStyle(@RequestParam("userId") Integer userId, @RequestParam("style") String style) {
        List<Clothing> clothingList = clothingMapper.selectByUserIdAndStyle(userId, style);
        return Result.success("获取成功", clothingList);
    }

    /**
     * 删除衣服
     */
    @DeleteMapping("/delete/{id}")
    public Result delete(@PathVariable Integer id) {
        System.out.println("开始删除衣服，ID: " + id);
        try {
            // 先获取衣服信息
            Clothing clothing = clothingMapper.selectById(id);
            System.out.println("获取到衣服信息: " + clothing);
            if (clothing == null) {
                System.out.println("衣服不存在，ID: " + id);
                return Result.error("衣服不存在");
            }
            
            // 打印clothing对象的信息
            System.out.println("Clothing ID: " + clothing.getId());
            System.out.println("Clothing FileName: " + clothing.getFileName());
            System.out.println("Clothing URL: " + clothing.getUrl());
            System.out.println("Clothing Type: " + clothing.getType());
            
            // 检查fileName是否为null
            if (clothing.getFileName() == null) {
                System.err.println("FileName is null for clothing with ID: " + id);
                // 直接删除数据库记录，因为Minio中没有对应的图片
                int result = clothingMapper.deleteById(id);
                System.out.println("删除数据库记录结果: " + result);
                if (result > 0) {
                    return Result.success("删除成功");
                } else {
                    return Result.error("删除失败");
                }
            }
            
            try {
                // 删除Minio中的图片
                System.out.println("开始删除Minio中的图片，fileName: " + clothing.getFileName());
                minioUtil.delete(clothing.getFileName(), "campus-outfit", "myclothes/");
                System.out.println("Minio图片删除成功");
            } catch (Exception e) {
                System.err.println("删除Minio图片失败: " + e.getMessage());
                // 即使Minio删除失败，也要继续删除数据库记录
            }
            
            // 删除数据库记录
            int result = clothingMapper.deleteById(id);
            System.out.println("删除数据库记录结果: " + result);
            if (result > 0) {
                return Result.success("删除成功");
            } else {
                return Result.error("删除失败");
            }
        } catch (Exception e) {
            System.err.println("删除衣服失败: " + e.getMessage());
            e.printStackTrace();
            return Result.error("删除失败：" + e.getMessage());
        }
    }

    /**
     * 获取衣服详情
     */
    @GetMapping("/detail/{id}")
    public Result detail(@PathVariable Integer id) {
        Clothing clothing = clothingMapper.selectById(id);
        if (clothing != null) {
            return Result.success("获取成功", clothing);
        } else {
            return Result.error("衣服不存在");
        }
    }
}
```

```javascript
const openWardrobeDialog = async () => {
  showWardrobeDialog.value = true
```

## 三、API接口文档

### 1. 上传衣服
- **URL**: `/api/wardrobe/upload`
- **Method**: POST
- **参数**:
  - `file`: 图片文件
  - `type`: 衣服类型（上衣、裤子、裙子、鞋子等）
  - `color`: 颜色
  - `style`: 风格（休闲、正式、运动等）
  - `season`: 季节（春季、夏季、秋季、冬季）
  - `description`: 描述
- **返回**: 上传成功信息和图片URL

### 2. 获取衣柜列表
- **URL**: `/api/wardrobe/list`
- **Method**: GET
- **参数**: `userId` (用户ID)
- **返回**: 衣服列表

### 3. 根据类型获取衣服
- **URL**: `/api/wardrobe/listByType`
- **Method**: GET
- **参数**:
  - `userId`: 用户ID
  - `type`: 衣服类型
- **返回**: 对应类型的衣服列表

### 4. 根据季节获取衣服
- **URL**: `/api/wardrobe/listBySeason`
- **Method**: GET
- **参数**:
  - `userId`: 用户ID
  - `season`: 季节
- **返回**: 对应季节的衣服列表

### 5. 根据风格获取衣服
- **URL**: `/api/wardrobe/listByStyle`
- **Method**: GET
- **参数**:
  - `userId`: 用户ID
  - `style`: 风格

### 6. 删除衣服
- **URL**: `/api/wardrobe/delete/{id}`

- **参数**: `id` (衣服ID)

### 7. 获取衣服详情
- **URL**: `/api/wardrobe/detail/{id}`

- **参数**: `id` (衣服ID)

## 四、功能说明

### 主要功能
1. **衣服上传**: 用户可以上传衣服图片到衣柜，支持设置类型、颜色、风格、季节和描述
2. **衣柜管理**: 查看、筛选和删除衣柜中的衣服
3. **分类查询**: 按类型、季节、风格查询衣服
4. **AI分析集成**: 在AI穿搭分析时可以从衣柜中选择衣服进行搭配

### 技术特点
- 使用Minio进行图片存储，生成预签名URL
- 支持多维度的衣服分类和查询
- 与AI分析模块无缝集成
- 完整的CRUD操作支持
