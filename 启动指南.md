# 校园穿搭推荐系统启动指南

## 项目结构
- **backend/**: Java Spring Boot后端服务
- **frontend/**: Vue 3 + TypeScript前端应用

## 环境要求

### 后端环境
- Java 17+
- Maven 3.6+
- MySQL 8.0+
- Redis 6.0+
- MinIO (对象存储服务)

### 前端环境
- Node.js 16+
- npm 8+

## 启动步骤

### 1. 环境准备

#### 1.1 数据库准备
1. 启动MySQL服务
2. 登录MySQL：`mysql -u root -p`
3. 执行数据库初始化脚本：
```sql
-- 执行backend/db/schema.sql文件
SOURCE e:\Profession\specialty practice2\campus-outfit-project\backend\db\schema.sql;
```
4. 确认数据库创建成功：`SHOW DATABASES;` 应该能看到 `campus_outfit` 数据库

#### 1.2 Redis准备
1. 启动Redis服务
2. 确认Redis服务运行在默认端口6379

#### 1.3 MinIO准备
1. 启动MinIO服务：
```bash
minio server D:\minio-data
```
2. 访问MinIO控制台：http://localhost:9000
3. 登录：用户名 `minioadmin`，密码 `minioadmin`
4. 创建存储桶：`campus-outfit`

### 2. 后端服务启动

#### 2.1 安装依赖
```bash
cd backend
mvn clean install
```

#### 2.2 配置检查
检查 `backend/src/main/resources/application.yml` 中的配置：
- 数据库连接信息
- Redis连接信息
- MinIO连接信息

#### 2.3 启动服务
```bash
# 方式1：使用Maven
mvn spring-boot:run

# 方式2：使用Java直接运行
java -jar target/campus-outfit-backend-0.0.1-SNAPSHOT.jar
```

服务将在 http://localhost:8080 启动

### 3. 前端应用启动

#### 3.1 安装依赖
```bash
cd frontend
npm install
```

**注意：如果遇到依赖冲突问题，可以尝试以下方法：**

1. **使用legacy模式安装**：
```bash
npm install --legacy-peer-deps
```

2. **清理并重新安装**：
```bash
rmdir /s /q node_modules
npm cache clean --force
npm install --legacy-peer-deps
```

3. **单独安装esbuild依赖**：
```bash
npm install esbuild@^0.19.0
npm install --legacy-peer-deps
```

4. **使用yarn代替npm**：
```bash
npm install -g yarn
yarn install
```

#### 3.2 启动开发服务器
```bash
npm run dev
```

前端应用将在 http://localhost:5173 启动

### 4. 访问系统
1. 打开浏览器访问：http://localhost:5173
2. 即可使用校园穿搭推荐系统

## 常见问题

### 后端启动失败
1. 检查数据库连接是否正确
2. 检查Redis是否正常运行
3. 检查MinIO服务是否启动且存储桶是否创建
4. 查看日志文件分析具体错误

### 前端无法访问后端接口
1. 检查前端配置中的API地址是否正确
2. 检查后端CORS配置是否允许前端域名访问
3. 检查后端服务是否正常运行

### 图片上传失败
1. 检查MinIO服务是否正常运行
2. 检查MinIO存储桶是否存在
3. 检查MinIO访问凭证是否正确
